
# Banking API Project

## Backend (Django)
- Models: BankAccount, Transaction
- APIs:
  - GET /api/accounts/
  - POST /api/send/

## Frontend (React)
- Sidebar + Navbar
- Pages:
  - Login
  - Dashboard
  - Send Money
  - Deposit
  - Transactions
  - Profile

## Run Backend
```bash
pip install django djangorestframework
python manage.py runserver
```

## Run Frontend
Use any React setup (Vite recommended)

---

## Features
- Simple banking transfer API
- Balance update logic
- Transaction history
- React UI structure
